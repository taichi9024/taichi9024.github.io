[github] コミットメッセージの変更方法

2021/2/28

コミット時のコミットメッセージをよくタイプミスしてその度にググっているので忘備録として。(書くほどのことでもない..)
git commit -m --amend "メッセージ"　以上。。。
amend 修正するという意味の英単語。
githubを使いこなしてより生産的に開発をしたい！
